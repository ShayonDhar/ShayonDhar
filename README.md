## 🚀 Featured Projects

### [🏠 Aelwyd — Welsh Property Affordability API](https://github.com/ShayonDhar/wales-mortgage-calculator)
> *Active Development — Phase 2 (React UI) in progress*

A REST API that calculates mortgage affordability and benchmarks buying power against **live HM Land Registry property data** for Welsh postcodes. Built with financial precision using `BigDecimal` arithmetic and Spring caching for performance.

**Highlights:**
- Applies FCA standard 4.5× loan-to-income lending caps
- Full month-by-month amortisation schedules
- Benchmarks against real Welsh property prices (CF, SA, LL, NP postcodes)
- Spring `@Cacheable` cuts repeated SQL queries from ~50ms → ~2ms

`Java 21` `Spring Boot 3.x` `PostgreSQL` `Docker` `Lombok` `Maven`

---

### [💎 JewelThieves](https://github.com/ShayonDhar/JewelThieves)
> *CS-230 Software Engineering Module @ Swansea University*

A fully functional Java recreation of the 1997 game *Jewel Chase* from the Microsoft Entertainment Pack. Built from scratch — no game frameworks — using JavaFX and custom ASCII-based level parsing.

**Highlights:**
- 3 unique enemy AI types (Flying Assassin, Floor Follower, Smart Thief with pathfinding)
- Dynamic items: colour-coded levers, proximity-triggered bombs, time-extension clocks
- Save/load game state, player profiles, and per-level top-10 leaderboards

`Java` `JavaFX` `FXML` `CSS` `OOP`
